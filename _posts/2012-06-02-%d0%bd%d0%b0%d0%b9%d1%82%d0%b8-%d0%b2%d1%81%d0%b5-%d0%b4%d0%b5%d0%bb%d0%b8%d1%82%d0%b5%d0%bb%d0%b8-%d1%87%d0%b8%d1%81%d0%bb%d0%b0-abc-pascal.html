Дано: число n

Написать программу, которая выводит все его делители.

 
Т. е. нам нужно найти такие числа Xi [1;n] при которых число n без остатка делиться на Xi.
var n,i:integer;
begin
readln(n);
for i:=1 to n do
if n mod i = 0 then write(i,' ');
readln;
end.</code>
