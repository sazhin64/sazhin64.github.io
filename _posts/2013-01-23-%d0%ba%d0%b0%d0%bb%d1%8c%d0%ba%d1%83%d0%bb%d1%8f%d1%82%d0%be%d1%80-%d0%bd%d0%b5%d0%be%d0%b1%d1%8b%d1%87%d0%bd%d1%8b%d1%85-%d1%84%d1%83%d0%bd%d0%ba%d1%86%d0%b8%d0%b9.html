Многие люди, когда учатся программированию, начинают с того, что создают калькулятор. Мы тоже создали не один, а теперь мне хотелось бы предложить вам создать калькулятор нечасто используемых функций. Этими функциями могут быть x^y, корень квадратный из x, корень из x n-й степени, экспонента числа x, число x в 16-й системе счисления, и т. д.
Кроме приведенных функций имеется еще множество встроенных, которые можно вычислить одной единственной операцией.
Для создания проекта на форме разместите 3 текстбокса и 7 командных кнопок. По желанию можете вставить лейблы для объяснений.
5 кнопок нам нужны для вычислений, 6-я и 7-я кнопки: очистка и выход.
печатаем программный код
Private Sub Command1_Click()
Text3.Text = Val(Text1.Text) ^ Val(Text2.Text)
End Sub

Private Sub Command2_Click()
Text3.Text = Sqr(Val(Text1.Text))
Text2.Text = ""
End Sub

Private Sub Command3_Click()
Text3.Text = Val(Text1.Text) ^ (1 / Val(Text2.Text))
Text2.Text = ""
End Sub

Private Sub Command4_Click()
Text3.Text = Exp(Val(Text1.Text))
Text2.Text = ""
End Sub

Private Sub Command5_Click()
Text3.Text = Hex(Val(Text1.Text))
Text2.Text = ""
End Sub

Private Sub Command6_Click()
Text1.Text = ""
Text2.Text = ""
Text3.Text = ""
End Sub

Private Sub Command7_Click()
End
End Sub
