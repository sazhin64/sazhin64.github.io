Проект «Расчет площади треугольника»
по формуле s = √(p*(p-a)*(p-b)*(p-c))
Где p = (a + b + c) / 2
Работа с формой: располагаем объекты Textbox – 4, Label – 4, CommandButton – 1.
В текстбоксах 1-3 будет записываться значение сторон треугольника, в лейблах записываются обозначения: A = ?, B = ?, C = ?, S = ? Командная кнопка – для расчета.
Код проекта:
Private Sub Command1_Click()
a = Val(Text1.Text)
b = Val(Text2.Text)
c = Val(Text3.Text)
p = (a + b + c) / 2
Text4.Text = (p * (p - a) * (p - b) * (p - c)) ^ (1 / 2)
End Sub</code>
