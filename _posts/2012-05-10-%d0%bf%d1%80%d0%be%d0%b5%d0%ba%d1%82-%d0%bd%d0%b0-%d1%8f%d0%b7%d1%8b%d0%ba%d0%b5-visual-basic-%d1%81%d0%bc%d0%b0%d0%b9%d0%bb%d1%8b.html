Проект на языке Visual Basic «Смайлы».
1. Работа с формой. Создать 2 рисунка смайла в программе Paint: один веселый, другой грустный. На форме разместить 2 объекта Image1, Image2. Свойство picture – bitmap, загрузить нужные рисунки
2. Программный код:

Private Sub Image1_Click()
Print "Улыбайся!"
End Sub

Private Sub Image2_Click()
Print "Хватит смеяться?"
End Sub</code>

Работа проекта: при нажатии на первый смайл появляется надпись: «Улыбайся!», при нажатии на второй смайл – «Хватит смеяться!»
Работа рассчитана на учащихся 3-4 класса, которым будет интересно выполнить рисунок в Paint, раскрасить его, поработать с формой, внести свою редакцию в цвет, размер шрифта, содержание выводимой фразы.
