      Таблица 2_Проверка

      1. Заполните таблицу как в образце. В поле проверка</strong> введите формулу: =ЕСЛИ(Вопросы!$B2="";"";ЕСЛИ(Вопросы!$B2="Верный</strong> ответ</strong>";"верно";"ошибка")). Логическое если</strong> проверяет, верно ли выбран ответ. В случае, если истина, «верно», если ложь, «ошибка».</li>
      2. В поле Балл </strong>введите формулу: =ЕСЛИ(B2="Верно";1;0). Формула выставляет один балл за правильный ответ и 0 баллов за неправильный.</li>

        Таблица 3_Проверка

        1. В поле балл</strong> введите формулу: =B2. Она будет отражать полученный балл. Если в таблице несколько вопросов, то нужно применить формулу =СЧЕТЕСЛИ(интервал ячеек с Баллами</strong>) и критерий</strong>. Формула посчитает количество баллов.</li>
        2. В поле Отметка введите формулу: логическое если</strong>, где положительное значение 1 – отметка «отлично», 0 – отметка «неудовлетворительно» =ЕСЛИ(B2=1;"отлично";"неудовлетворительно") Вставляйте свои номера ячеек!</strong></li>
        3. Лист проверка можно скрыть и защитить паролем, тогда он будет недоступен для тех, кто выполняет тест.</li>
        4. Для повторного использования теста можно использовать макрос Очистка</strong>. Для этого на лист Вопросы.</strong> Создание кнопки очистка</strong>. Путь: Вид</strong> – макрос</strong> – макросы</strong> – войти</strong>. Откроется редактор макросов работающий в среде Visual Basic. Написать код:</li>
          </ol>
          Sub Очистка()

          Range("A14").Select

          Range("A14").FormulaR1C1 = Empty

          End Sub</code>

          Сохранить. Для кнопки очистка</strong> выбрать путь: файл</strong> – параметры </strong>Excel</strong> – поставить галочку Показывать вкладку «Разработчик на ленте»</strong>, ОК</strong>. Разработчик – вставить – кнопка</strong>.

          1. Поставьте курсор в нужное место и создайте кнопку нужного размера. Автоматически появится окно с макросами. Выберите макрос Очистка</strong>. Ок. Правой кнопкой мыши щелкнуть на кнопку,</strong> сделать надпись Очистка</strong>. Макрос сохранить в текущей книге.</li>
          2. Сохранить файл, проверить работоспособность.</li>
          3. Создавая свои тесты, используйте картинки, заливку, формулы и т д.</li>
