Проект «Светофор».
1. Работа с формой. Нарисовать в Пэйнте 3 смайла, поместить в отдельную папку.
Расположить на форме объекты: Image – 3, Option – 3, поменять свойства (Backcolor, Caption), при загрузке картинки не забудьте, что у нее есть свое полное имя, отличающееся от того, что написано у меня. В коде программы укажите полный путь до картинки, который можно найти, нажав на картинку правой кнопкой мыши.
Private Sub Option1_Click()
Image1.Visible = True
Image2.Visible = False
Image3.Visible = False
Image1.BorderStyle = 1
Image1.Picture = LoadPicture("C:\Users\Таня\Desktop\Папки с раб.стола\Сергей\Новая папка (5)\Светофор\Новая папка\грусть.jpg")
End Sub

Private Sub Option2_Click()
Image2.Visible = True
Image1.Visible = False
Image3.Visible = False
Image2.BorderStyle = 1
Image2.Picture = LoadPicture("C:\Users\Таня\Desktop\Папки с раб.стола\Сергей\Новая папка (5)\Светофор\Новая папка\норма.jpg")
End Sub

Private Sub Option3_Click()
Image3.Visible = True
Image1.Visible = False
Image2.Visible = False

End Sub</code>
