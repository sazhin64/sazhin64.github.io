Проект на VBA для Excel «Калькулятор».</p>
Проект родился при разборе материала из учебника «Информационная культура» авторов Т.А. Матвеевой и А.Г. Гейн, § 2. Первый проект на VBA. В нем приводится пример с вычислением суммы двух чисел посредством написания макроса для командной кнопки.

Рис. 1. Описание проекта в учебнике.

В учебнике дано описание одного арифметического действия. Нетрудно догадаться, что написав подобным образом кнопки для оставшихся действий, мы получим простой калькулятор в Excel. Кроме этого, можно добавить функции извлечения корня квадратного и еще самые разнообразные функции.

Этапы создания проекта.

1. Работа с ячейками.

Рис. 2. Лист Excel c выделенными ячейками.

Выделите необходимое количество ячеек, и отформатируйте (измените цвет, размер шрифта) по своему желанию. В ячейки между будущими полями для ввода чисел вставьте символы арифметических действий.

2. Создайте 4 командные кнопки Command button. Зайдите в свойства кнопок и поменяйте имя формы, например, CommandButton1 на Plus. В свойстве Caption напишите нужное действие, например «Сложение». Задайте цвет фона и надписи, измените размер шрифта.

3. Написание макроса.

Щелкните два раза по кнопке в режиме конструктора и у вас откроется окно редактора Visual Basic. Напишите последовательно код всех кнопок. Приведу пример одной из них. Остальные сделайте аналогично первой.

Private Sub plus_Click()

Dim a As Single

Dim b As Single

If TypeName(Cells(4, 3).Value) <> "Double" Then GoTo ex

If TypeName(Cells(4, 6).Value) <> "Double" Then GoTo ex

a = Cells(4, 3)

b = Cells(4, 6)

c = a + b

Cells(10, 2) = c

Exit Sub

ex: MsgBox ("Вы ввели не числа! Повторите ввод")

End Sub
</code>
Этот код предназначен для кнопки «Сложение». Код мог бы быть короче, но при этом, при неправильном формате ввода данных, программа будет выдавать ошибку.

Для тех, кто не смог разобраться и выполнить самостоятельно, приведу полный код программы, но не торопитесь его списывать, подумайте.

Можете добавить и другие кнопки. Такой калькулятор выгодно отличается от обычного тем, что при введении двух чисел, с ними можно совершить несколько арифметических действий не вводя их повторно.

Если не можете, то вот полный код:
