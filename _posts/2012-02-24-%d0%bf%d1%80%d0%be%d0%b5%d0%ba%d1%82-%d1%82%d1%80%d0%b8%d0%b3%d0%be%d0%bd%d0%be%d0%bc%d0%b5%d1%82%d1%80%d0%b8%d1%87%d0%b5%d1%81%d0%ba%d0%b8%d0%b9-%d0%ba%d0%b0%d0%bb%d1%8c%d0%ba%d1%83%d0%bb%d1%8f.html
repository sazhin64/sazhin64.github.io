Проект «тригонометрический калькулятор» на VB</p>
Работа с формой: на форме расположить 6 командных кнопок: 4-для тригонометрических функций, одна для очистки, одна для выхода из программы. 4 текстбокса: 3-для ввода градусов, минут, секунд и один для вывода результата. И еще 4 лейбла: для надписей над текстбоксами и окном результата (градусы, минуты, секунды, результат).

Придаем форме оригинальность путем изменения цвета кнопок, размера и типа шрифта, загружаем на форму картинку. Лейблы делаем прозрачными. Вся эта работа производится в окне свойств (Properties).

  1. Далее, определяем тип переменных, которые будут вводиться в текстбоксы:</li>
    </ol>
    Const Pi = 3.14

    Dim grad As Single

    Dim min As Single

    Dim sec As Single

    Dim angle As Single</code>

    1. Переименовываем Command1 в cmdSin (свойство Name) и, соответственно остальные кнопки кроме 5-й и 6-й. Код для первой кнопки дан целиком. Остальные пишутся аналогично.</li>

      Private Sub cmdSin_Click()

      grad = Text1.Text’ Для текстбоксов пишем код

      min = Text2.Text

      sec = Text3.Text

      angle = (grad + min / 60 * 100 + sec / 60 * 100) * Pi / 180’ Пишем функцию вычисления синуса угла

      Text4.Text = Sin(angle)’ В 4-й текстбокс выводим значение синуса для данного угла

      End Sub

      Результат для кнопки тангенса:

      Text4.Text = Tan(angle)

      Результат для кнопки косинуса:

      Text4.Text = Cos(angle)

      Результат для кнопки котангенса

      Text4.Text = 1 / Tan(angle)

      Код кнопки очистки:

      Private Sub Command5_Click()

      Text1.Text = ""

      Text2.Text = ""

      Text3.Text = ""

      Text4.Text = ""

      End Sub</code>

      Код кнопки выхода пишем самостоятельно.
